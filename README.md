## Demos Branch Layout

The `demos` branch is organized by topic. The table below summarizes each demo
folder at a high level.

| Folder | Description |
|--------|-------------|
| [`docker-demos`](../../tree/demos/docker-demos) | Docker-based examples for Variscite platforms. |
| [`machine-learning-demos`](../../tree/demos/machine-learning-demos) | Machine learning demos using frameworks such as TensorFlow Lite and Arm NN. |
| [`opencl`](../../tree/demos/opencl) | OpenCL examples focused on compute workloads for CPU and GPU. |
| [`tpm-demos`](../../tree/demos/tpm-demos) | TPM integration demo sources and related example code. |
| [`vscode-demos`](../../tree/demos/vscode-demos) | VS Code examples for development and debugging workflows. |
